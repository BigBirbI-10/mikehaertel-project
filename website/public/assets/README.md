## File Naming Convention

### Recommended Format:
```
[category]-[subject]-[description]-[date].jpg
```

### Examples:
```
astro-milkyway-austin-2024-08.jpg
project-homelab-server-rack-2025.jpg
headshot-professional-outdoor-2025.jpg
farescout-dashboard-screenshot-2025-01.jpg
pedicab-acl-festival-2024.jpg
```

### Guidelines:
- Use lowercase
- Separate words with hyphens
- Include date (YYYY-MM or YYYY-MM-DD) when relevant
- Be descriptive but concise
- Use common image formats: `.jpg`, `.png`, `.webp`

## Image Optimization Tips

Before uploading photos for web use:
1. **Resize for web**: 1920px wide for full-width images, 800px for thumbnails
2. **Compress**: Use tools like ImageOptim, TinyPNG, or Squoosh
3. **Format**: JPG for photos, PNG for graphics with transparency
4. **WebP**: Consider WebP format for better compression (with JPG fallback)
